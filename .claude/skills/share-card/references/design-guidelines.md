# 设计规范

Agent 生成卡片 HTML 时应遵循的设计标准。

## 色彩方案

### 暗色（默认）

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景主色 | `#0d1117` | card-wrapper 背景 |
| 背景副色 | `#161b22` | 渐变终点、卡片内区块 |
| 文字主色 | `#e6edf3` | 正文、标题 |
| 文字副色 | `#c9d1d9` | 次要正文 |
| 弱化文字 | `#8b949e` | 说明、时间戳、副标题 |
| 极弱文字 | `#484f58` | 水印、装饰文字 |
| 蓝色强调 | `#58a6ff` | 链接、高亮、用户标签 |
| 绿色强调 | `#7ee787` / `#238636` | 成功、AI 标签 |
| 紫色强调 | `#bc8cff` | 渐变、装饰光晕 |
| 边框色 | `#30363d` | 分割线、卡片边框 |

## 字体层级

| 层级 | 字号范围 | 粗细 | 用途 |
|------|----------|------|------|
| Hero | 52-80px | 700-800 | 海报主标题 |
| H1 | 36-52px | 600-700 | 卡片标题 |
| H2 | 24-30px | 600 | 区块标题 |
| Body | 18-24px | 400 | 正文 |
| Small | 14-16px | 400 | 说明、时间戳 |
| Code | 16-20px | 400 | 代码块 |

## 间距

- 卡片外边距（padding）：60-120px
- 最小安全边距：40px（防止内容被裁切）
- 区块间距：24-50px
- 文字行高：1.5-1.7

## 光晕效果

用伪元素 + radial-gradient 增加层次感，避免扁平：

```css
#card-wrapper::before {
  content: '';
  position: absolute;
  /* 位置和大小根据需要调整 */
  background: radial-gradient(ellipse, rgba(88,166,255,0.08), transparent 70%);
  pointer-events: none;
}
```

- 不透明度控制在 0.06-0.15 之间
- 一到两个光晕叠加（`::before` + `::after`）
- 位置不对称（如左上 + 右下）

## 自适应模式快速设计决策

| 内容特征 | 推荐方向 |
|----------|----------|
| 问答对话 | 气泡隐喻，左右交替 |
| 叙事长文 | 杂志风，大标题 + 分栏 |
| 代码为主 | 终端/IDE 风 |
| 数据/列表 | 卡片网格 |
| 推理链/过程 | 时间线/步骤条 |
| 金句/结论 | 大字报 |
| 技术教程 | 文档风，步骤编号 |
