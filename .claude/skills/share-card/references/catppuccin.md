# Catppuccin 色板

来源：[Catppuccin](https://github.com/catppuccin/catppuccin) (MIT, 18k+ stars) — 4 套粉彩主题，每套 26 色。

Agent 写自适应卡片时，根据内容类型选一套 flavor，直接取色。

## 四套 Flavor 速查

| Flavor | 基调 | Base 底色 | Text 文字色 | 适合 |
|--------|------|-----------|------------|------|
| **Latte** 🌻 | 暖浅 | `#eff1f5` | `#4c4f69` | 教程、文档、浅色风 |
| **Frappé** 🪴 | 中暗 | `#303446` | `#c6d0f5` | 通用暗色 |
| **Macchiato** 🌺 | 暗 | `#24273a` | `#cad3f5` | 技术分析、代码 |
| **Mocha** 🌿 | 最暗 | `#1e1e2e` | `#cdd6f4` | 终端风、金句海报 |

## 使用指南

**写自适应 HTML 时：**

1. 选 flavor（见下）
2. `#card-wrapper { background: <Base>; color: <Text>; }`
3. 副标题/次要文字用 `Subtext0`
4. 卡片/区块背景用 `Surface0`
5. 强调色从 14 个 accent 中选（Green/Blue/Peach 等）
6. 边框/分割线用 `Overlay0`

### Flavor 选择

| 内容类型 | 推荐 Flavor | 推荐强调色 |
|----------|------------|-----------|
| 教程/指南 | Latte | Blue |
| 技术分析 | Macchiato | Teal 或 Green |
| 代码展示 | Mocha | Green 或 Blue |
| 金句/结论 | Mocha | Peach 或 Lavender |
| 对话/讨论 | Frappé | Mauve + Green |
| 推理/时间线 | Macchiato | Peach |
